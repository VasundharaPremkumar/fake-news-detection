# Fake News Detection

Small project that trains a TF-IDF + Logistic Regression classifier to detect fake news.
This repo contains the data-processing and training scripts and a small prediction script.

## Structure
- `data/` — raw datasets (ignored from git; keep local)
- `models/` — saved model & vectorizer (ignored from git)
- `src/` — code (`fakenews.py`, `train_and_save.py`, `predict.py`, `app.py`)
- `README.md` — this file
- `requirements.txt` — Python dependencies
